# 2.0 บทนำ

> "เงินเป็นหลักประกันว่าเราอาจมีสิ่งที่เราต้องการในอนาคต แม้ว่าเราจะไม่ต้องการอะไรในขณะนี้ แต่มันก็ทำให้มั่นใจได้ว่าเราจะสามารถตอบสนองความต้องการใหม่ๆ ได้เมื่อมันเกิดขึ้น"\
> **Aristotle,อริสโตเติล**
